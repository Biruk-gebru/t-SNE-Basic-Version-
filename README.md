# t-SNE Basic Version - Non-linear Dimensionality Reduction

This repository contains a basic implementation of the t-SNE (t-Distributed Stochastic Neighbor Embedding) algorithm using Python. It includes the source code, a Jupyter notebook for visualization, and the final report.

## Files
- `tsne.py`: Python implementation of the t-SNE algorithm.
- `tsne.ipynb`: Jupyter Notebook demonstrating the usage and visualization.
- `t-SNE_Basic_Version_Non-linear_Dimensionality_Reduction_for_Visualization.pdf`: The final project report.
